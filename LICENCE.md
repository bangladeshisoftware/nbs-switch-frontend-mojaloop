### About This Project

This project implements the **Mojaloop Switch Portal** — the administrative and
operational web interface for Hub Operators to manage the Mojaloop Switch. It
provides dashboards, participant management, settlement monitoring, and system
configuration capabilities.

**Key Features:**
- Hub Operator Dashboard
- Participant (DFSP) Management
- Settlement Window Management
- Transaction Monitoring
- System Health and Status

**Listed in Mojaloop Service Provider Directory:**
https://mojaloop.io/service-provider-directory/entry/1955/
